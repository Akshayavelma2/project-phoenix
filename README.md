# Project Phoenix

## Project Vision
Project Phoenix is an EdTech platform designed to help learners gain
industry-ready skills through structured courses and hands-on projects.

## Goal
To build a secure, scalable, and continuously deployable learning platform
using modern DevOps practices.

## Planned Tech Stack
- Frontend: React
- Backend: Node.js
- Database: PostgreSQL
- CI/CD: Jenkins
- Version Control: Git
